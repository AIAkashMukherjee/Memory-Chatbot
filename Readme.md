# 🧠 Memory-Enhanced AI Chatbot

An **interactive**, **document-aware** chatbot with persistent memory, powered by OpenAI & Groq LLMs + Pinecone (with optional local Chroma fallback). Upload PDFs, ask questions, and get context-rich answers—all in one sleek UI!

---

## 🚀 Features

- **📄 Document Upload & Indexing**— Drop in PDFs and instantly index their contents for conversational retrieval.
- **🗂️ Retrieval-Augmented Generation (RAG)**— Pinecone vector store (or local Chroma) finds the most relevant text chunks.
- **💭 Persistent Memory**— Remembers your last _k_ messages (configurable!) for coherent multi-turn dialogue.
- **⚡ Groq + OpenAI Fallback**— If you skip the OpenAI key, it auto-defaults to lightning-fast Groq LLM.
- **🔧 Streamlit Interface**— Beautiful, reactive chat UI: upload docs, chat, clear memory, save/load sessions.
- **🛠️ Docker-Ready**
  — One-line Docker build & run for seamless deployment.

---

## 📦 Getting Started

### Prerequisites

- **Docker** (optional, for containerized run)
- **Python 3.11+** (if running locally)
- **API Keys**
  - `OPENAI_API_KEY` (optional — for GPT-3.5/4 generation)
  - `PINECONE_API_KEY` & `PINECONE_ENV` (for vector database)
  - _Groq key baked in as fallback — no action needed!_

## ⚙️ Configuration & Usage

* **Initialize Chatbot** : Enter your OpenAI key (optional) in the sidebar and click “Initialize Chatbot”.
* **Upload PDFs** : Use the “Upload Document” widget to index new files.
* **Chat** : Type in the chat box—bot will pull context from your docs & past messages.
* **Clear Memory** : Click “Clear Memory” to reset the conversation history.
* **Save/Load Memory** : (Coming soon in UI!) Persist your session to disk or restore it later.
